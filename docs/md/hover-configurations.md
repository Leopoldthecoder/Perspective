# Configurations
Compared to Scroll, Hover has much simpler configs.

## max
- Type: `Number`
- Default: `20`

## reverseTilt
- Type: `Boolean`
- Default: `false`

## perspective
- Type: `Number`
- Default: `1000`

## easing
- Type: `String`
- Default: `cubic-bezier(.03, .98, .52, .99)`

## scale
- Type: `Number`
- Default: `1`

## speed
- Type: `Number`
- Default: `300`

## disabledAxis
- Type: `String`
- Default: `''`

## reset
- Type: `Boolean`
- Default: `true`

## layers
- Type: `Object[]`
- Default: `[]`

### multiple
- Type: `Number`
- Default: `0.2`

### reverseTranslate
- Type: `Boolean`
- Default: `false`
